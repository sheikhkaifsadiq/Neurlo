# Neurlo — AI That Thinks Before You Ask

Neurlo is a next-generation AI operating system designed for the professional workflow. It predicts needs, drafts actions, and connects your digital tools into a single cognitive unit.

## 🚀 Features & UI Architecture

*   **Predictive Context:** Surfaces relevant documents, threads, and data seamlessly.
*   **Instant Actioning:** Drafts emails, Slack responses, and task updates based on real-time triggers.
*   **Visual Workflows:** Drag-and-drop editor for crafting custom agentic automations.
*   **Modern Interface:** Built with Next.js (App Router), Tailwind CSS, and Shadcn UI for a stunning, responsive, and highly interactive user experience.

## 🛠 Tech Stack Overview

*   **Frontend:** Next.js 14, React, Tailwind CSS, Lucide Icons
*   **UI Components:** Radix UI, Shadcn
*   **Database Schema:** Prisma ORM, PostgreSQL (with pgvector for RAG)
*   **AI & Caching (Dependencies):** OpenAI, Upstash Redis, QStash

## 🔒 Security & Architecture Note

This public repository serves as a demonstration of the platform's architecture, UI/UX, and core capabilities. For security and proprietary reasons, sensitive business logic, payment gateway webhooks, and proprietary AI algorithms are maintained in a separate private repository which is deployed to production.
